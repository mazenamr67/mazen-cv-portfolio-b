Project Overview

This is my submission for Phase 2 of the Google AI Front-End Developer Assessment.

In Phase 1, I built a fully static portfolio from scratch using only HTML, CSS, and JavaScript. I scored 90% as a solo submission.

For Phase 2, I rebuilt the portfolio as a dynamic web application. All content is now loaded from a single data.json file. I also added many new visual effects and interactions while keeping the same color scheme and navigation from Phase 1.

All new code and techniques added in Phase 2 are marked with // new: or <!-- new: --> comments.

AI Tools Used

- Claude (Anthropic) - main tool used throughout the project for generating code, fixing bugs, explaining unfamiliar concepts, and reviewing my work.
- Grok (xAI) - used early on to get initial project structure ideas and to double-check some of the code Claude produced.

Main Features Added in Phase 2
